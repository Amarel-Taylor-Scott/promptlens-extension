## Description

### What it does
PromptLens reverse-engineers AI-generated images into detailed, usable prompts. Right-click any image on the web, and PromptLens will analyze it and generate prompts optimized for Midjourney, DALL-E, Stable Diffusion, and Flux — all in seconds.

### How it works
1. Right-click any image on any website
2. Select "Generate Prompt with PromptLens"
3. A new tab opens showing the generated prompts as they're created
4. Copy the prompt and use it in your favorite AI image generator

### Features
- One-click prompt extraction from any image on the web
- Prompts optimized for 4+ AI models (Midjourney, DALL-E, Stable Diffusion, Flux)
- Instant results — the page opens immediately while prompts generate
- Duplicate detection — previously analyzed images return results instantly
- Generation history in the extension popup
- Settings page for customizing behavior

### Who it's for
- AI artists who want to learn from other images
- Prompt engineers refining their craft
- Creators looking for inspiration
- Anyone curious about how an AI image was made

### Free to use
3 free generations per day with no account needed. Link your email for 5/day, or sign up at promptlens.polsia.app for unlimited access.

### Privacy
PromptLens only processes images you explicitly choose to analyze. Images are processed temporarily for prompt extraction and are not stored permanently. No browsing data is collected. See our full privacy policy at https://promptlens.polsia.app/privacy
