## Detailed Description

PromptLens reverse-engineers AI-generated images into detailed, usable prompts. Right-click any image on the web, and PromptLens will analyze it and generate prompts optimized for Midjourney, DALL-E, Stable Diffusion, and Flux — all in seconds.

HOW IT WORKS
1. Right-click any image on any website
2. Select "Generate Prompt with PromptLens"
3. A new tab opens showing the generated prompts as they're created
4. Copy the prompt and use it in your favorite AI image generator

FEATURES
• One-click prompt extraction from any image on the web
• Prompts optimized for 4+ AI models (Midjourney, DALL-E, Stable Diffusion, Flux)
• Instant results — the page opens immediately while prompts generate
• Duplicate detection — previously analyzed images return results instantly
• Generation history in the extension popup
• Settings page with authentication options and model preferences

WHO IT'S FOR

• Prompt engineers refining their craft
• Creators looking for inspiration
• Anyone curious about how an AI image was made

## Permission Justifications (required by Chrome)

### <all_urls> (host permissions)
"Required to fetch images from any website when the user explicitly right-clicks an image and selects 'Generate Prompt with PromptLens'. The extension only accesses image data from URLs the user chooses to analyze — it never accesses any other page content or browsing data."

### contextMenus
"Adds the 'Generate Prompt with PromptLens' option to the right-click context menu when clicking on images."

### storage
"Stores user preferences (authentication mode, default models, behavior settings) and a local history of recent generations. All data stays in the browser."

### notifications
"Shows a system notification as a fallback when the on-page toast notification cannot be displayed (e.g., on chrome:// pages or restricted tabs)."

### scripting
"Injects a small toast notification script into the active tab to show generation progress (uploading, success, error) directly on the page the user is browsing."

### activeTab
"Allows the extension to interact with the current tab when the user triggers the context menu action."
