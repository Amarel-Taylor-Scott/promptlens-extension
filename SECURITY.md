# Security Policy

## Supported versions

Security fixes are applied to the latest `master` and the current store release.

| Version | Supported |
|---------|-----------|
| latest `master` / current store release | ✅ |
| older packaged releases | ❌ |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Email **amarel.taylor.s@gmail.com** with a description, reproduction steps, and
any suggested fix. You can expect an acknowledgement within a few days, after
which we'll coordinate a fix and disclosure.

## Scope

This extension processes only images you explicitly right-click, stores settings
locally in the browser, and talks to the PromptLens API. It collects no browsing
history. Please report any behavior that contradicts this, any excessive
permission, or any data-handling concern via the email above. See
[privacy-policy.html](privacy-policy.html) for the full privacy policy.
